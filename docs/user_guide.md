# Руководство пользователя CryptoSafe Manager

## Запуск

Из исходников:

```powershell
python run.py
```

Готовая сборка:

```text
dist\CryptoSafeManager\CryptoSafeManager.exe
```

## Создание базы

1. Откройте приложение.
2. Выберите создание новой базы.
3. Введите мастер-пароль.
4. Проверьте подсказки надёжности пароля.
5. При необходимости измените путь к файлу базы.
6. Нажмите `Создать`.

Имя файла по умолчанию: `CryptoSafe.db`.

Требования к мастер-паролю:

- минимум 8 символов;
- хотя бы одна заглавная буква;
- хотя бы одна цифра;
- спецсимвол рекомендуется, но не обязателен.

## Открытие базы

Используйте `Файл -> Открыть` и выберите `.db` файл. После выбора файла приложение попросит мастер-пароль.

## Записи

В главном окне можно:

- добавить запись;
- изменить запись;
- удалить запись;
- найти запись через строку поиска;
- скопировать логин, пароль, URL или пару `логин:пароль`.

Записи можно группировать по категориям.

## Смена мастер-пароля

Откройте `Файл -> Сменить пароль`.

Нужно указать:

- текущий мастер-пароль;
- новый мастер-пароль;
- подтверждение нового мастер-пароля.

Для нового пароля показывается надёжность и список невыполненных требований. После смены пароля записи перешифровываются новым ключом.

## Блокировка

В настройках можно выбрать:

- не блокировать при потере фокуса;
- блокировать сразу;
- блокировать через заданное время.

Блокировка срабатывает при:

- сворачивании окна;
- Alt+Tab;
- клике в другое приложение.

Если включён трей, приложение скрывается в трей. Когда пользователь снова открывает окно, появляется ввод мастер-пароля для последней открытой базы.

## Импорт

Откройте `Файл -> Импорт`.

Поддерживаются:

- encrypted JSON;
- CSV;
- Bitwarden JSON;
- LastPass CSV.

Для encrypted JSON выберите способ расшифровки: пароль или приватный `.pem` ключ. Перед импортом показывается предпросмотр записей и категорий.

## Экспорт

Откройте `Файл -> Экспорт`.

Можно выбрать:

- формат экспорта;
- отдельные записи;
- категории;
- способ шифрования: пароль или публичный `.pem` ключ получателя.

## Sharing и QR

Функция sharing создаёт защищённый пакет записи и показывает QR-код. Срок действия задаётся в минутах.

## Panic Mode

Panic mode используется для быстрого скрытия и блокировки чувствительных данных.

Активация:

- `Ctrl+Shift+Q`;
- пункт меню в системном трее.

## Трей

Иконка в трее показывает состояние приложения:

- заблокировано;
- открыто;
- panic mode.

Через меню трея можно показать окно, заблокировать базу, открыть базу, очистить буфер обмена и включить panic mode.

## Отправка приложения другому человеку

Сначала соберите приложение:

```powershell
python -m PyInstaller CryptoSafeManager.spec --noconfirm
```

Потом архивируйте папку:

```text
dist\CryptoSafeManager
```

Получатель распаковывает архив и запускает `CryptoSafeManager.exe`.
